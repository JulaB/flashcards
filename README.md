# FlashCards app

This project is a simple flashcards application.

## Technologies
* Ruby 2.6.0
* Rails 5.2.2
* Postgres 11.1
* React 16.8.3
* Redux 4.0.1
* Webpack 4.29.3
* Node 10.15.0
* PostCSS
* Minitest
* Jest
* Enzyme

## Project Status
Work in progress.

## Install
* yarn install
* bundle install
* rails db:setup
* rails test
* yarn test

## Development
To run server in development with hot module replacement `overmind s -f Procfile.dev`

## Running the tests
* rails test
* yarn test
